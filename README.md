# Soluções Rosa — Landing Page

Landing page profissional da **Soluções Rosa**, empresa especializada em instalações elétricas, energia solar fotovoltaica e estruturas de telhado em Porto Alegre - RS.

## 🏢 Sobre a Empresa

- **Nome:** Soluções Rosa
- **Localização:** Porto Alegre - RS
- **Experiência:** 5+ anos no mercado
- **Projetos:** 1.000+ módulos fotovoltaicos instalados
- **Clientes:** 500+ clientes satisfeitos
- **Instagram:** [@solucoes.rosa](https://instagram.com/solucoes.rosa)

## 🔧 Serviços

| Serviço | Descrição |
|---------|-----------|
| Instalação Elétrica | Residencial e comercial com segurança e precisão |
| Energia Solar Fotovoltaica | Instalação de módulos para economia e sustentabilidade |
| Estruturas de Telhado | Construção, reparo, telhas e impermeabilização |
| Manutenção Elétrica | Preventiva e corretiva |
| Serviços Residenciais | Soluções completas para residências |
| Serviços Empresariais | Atendimento especializado para empresas |

## 📄 Estrutura da Página

1. **Navbar** — Logo, links de navegação e botão WhatsApp
2. **Hero** — Frase de destaque, subtítulo e CTA para WhatsApp
3. **Sobre** — Histórico da empresa e estatísticas
4. **Serviços** — Cards com os 6 serviços oferecidos
5. **Diferenciais** — 5 motivos para escolher a empresa
6. **Portfólio** — Galeria de trabalhos realizados
7. **Depoimentos** — Avaliações de clientes satisfeitos
8. **CTA Final** — Chamada para ação com botão WhatsApp
9. **Rodapé** — Contato, redes sociais e navegação
10. **Botão flutuante** — WhatsApp fixo no canto inferior direito

## 🎨 Design

- **Paleta:** Azul escuro (primária), dourado/amarelo (destaque), branco e cinza
- **Fontes:** Montserrat (títulos) e Open Sans (corpo)
- **Animações:** Fade-in com scroll (Intersection Observer)
- **Responsivo:** Otimizado para mobile e desktop

## 🛠 Tecnologias

- React 18 + TypeScript
- Vite 5
- Tailwind CSS v3
- Lucide React (ícones)
- shadcn/ui (componentes base)

## 🚀 Como rodar

```bash
npm install
npm run dev
```
